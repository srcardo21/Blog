ING Software y Datos

Tipos de Software

Software del Sistema

Son los Software que interactúan directamente con el usuario, algunos de estos permiten que se ejecuten otro tipo de aplicaciones. En esta categoría se pueden encontrar sistemas operativos como Windows, Linux, Android, Mac OS, y controladores de dispositivos y servidores.

Software de Programación

Son los que permiten que los programadores desarrollen código, a este grupo pertenecen, por ejemplo, los lenguajes de programación, compiladores y depuradores.

Software de Aplicaciones

Son todos los programas realizados por los usuarios, para realizar tareas específicas. Aquí es donde están la gran mayoría de aplicaciones que conocemos y que funcionan en diferentes dispositivos:

◉ Procesadores de texto y hojas de cálculo como: Word, Excel, Bloc de notas.

◉ Plataformas web, redes sociales, tiendas virtuales, sistemas de reservas, catálogos, sitios informativos.

◉ Videojuegos, Software a medida, App para dispositivos móviles, Aplicaciones con IA

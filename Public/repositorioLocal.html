Repositorio Local

Un repositorio local es una copia de un repositorio de GitHub almacenada en tu computadora.
Es como una copia de la versión actual del proyecto que puedes manipular, realizar cambios. experimentar y trabajar de forma independiente.

1. Nos vamos a ubicar en la carpeta donde esta el proyecto
2. Damos un Click derecho y seleccionamos la opción Open Git Bash Here

Comandos

Es necesario tener claro las principales características del software que desarrollamos, estas son definidas al momento de levantar los requerimientos y necesidades que debe cubrir.
Dependiendo de la magnitud del software, este necesitará, más o menos prioridad para determinadas características.

Usabilidad -> Que sea fácil de operar.
Flexibilidad -> Que sea fácil de modificar por los desarrolladores.

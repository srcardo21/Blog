Etapas

Para realizar un desarrollo adecuado, se deben considerar diferentes etapas que nos permiten estructurar adecuadamente el producto y enfocar exitosamente la construcción. A estas etapas se les conocen como Ciclo de vida del software y varían dependiendo el modelo de desarrollo que usa cada organización.

1. Etapa de análisis de requisitos.
2. Etapa de diseño.
3. Etapa de desarrollo.
4. Etapa de pruebas o verificación.
5. Etapa de documentación.
6. Etapa de mantenimiento.

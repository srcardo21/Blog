Branches

En Git, una rama es una línea de desarrollo independiente de un proyecto, que permite trabajar diferentes versiones del código sin afectar la rama principal y luego unirlas.

1. Si usamos el comando git branch, podemos ver las ramas que tenemos creadas.

2. Para crear una rama usamos el comando git branch nombreRama, donde después del branch, ponemos el nombre que tendrá la rama.

3. Si usamos nuevamente el comando git branch, podemos ver la rama que creamos.

4. Para cambiar entre una rama y otra usamos el comando git checkout nombreRama, donde después del checkout ponemos el nombre de la rama que vamos a usar.

◉ Adicional podemos cambiar el nombre de la rama con el comando git branch -m nombreActual NombreNuevo.

◉ O eliminar una rama, git branch -d nombreRama.

Nota: Para eliminar una rama, NO podemos estar sobre la misma.
